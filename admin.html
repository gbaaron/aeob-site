    Shop

    Manage merch products. Stored in the Airtable MerchProducts table.

    Required Airtable fields for the MerchProducts table

    Create these columns in Airtable if you haven't already:

    • Name — single-line text
    • Slug — single-line text
    • Description — long text
    • Price — number (currency)
    • ComparePrice — number (currency, for "was" pricing)
    • Category — single select: apparel, caps, accessories, throwback, new
    • Badge — single-line text (e.g. "New", "Bestseller")
    • Color — single-line text (hex, e.g. #1a1f5e)
    • Stock — number
    • Status — single select: Active, Draft, Archived
    • Sizes — single-line text, comma-separated (e.g. "S,M,L,XL")
    • SortOrder — number
    • ImageURL — URL

    Predictions

    Create prediction questions, then mark the correct answer when the episode airs.

    Required Airtable fields for the Predictions table
    • Question — long text
    • Status — single select: Open, Resolved, Closed
    • Options — long text (JSON array of strings)
    • CorrectAnswer — single-line text
    • Episode — single-line text (optional)
    • HostPicks — long text (JSON of {host: pick})
    • ResolvedAt — date
