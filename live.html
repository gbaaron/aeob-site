📺 AEOB stream setup

Netlify can't host RTMP ingest — you need a streaming provider in between. Recommended: Cloudflare Stream Live.

  1. Cloudflare Dash › Stream › Live Inputs › Create.
  2. Copy the RTMPS URL and Stream Key.
  3. In StreamYard › Custom RTMP destination, paste both.
  4. Copy the Live Input's Iframe URL (looks like https://customer-xxx.cloudflarestream.com/<uid>/iframe).
  5. Paste it below — the AEOB tab plays it natively.
Not configured

Also works with Mux, AWS IVS, Bunny Stream, or any provider that gives you an iframe embed / HLS URL.
